### Identifiers

* suffix ``_min`` and ``_max``
* suffix ``_0``, ``_1``, etc.
* prefix greek letters 
* ``alpha``, ``beta``, ``gamma``, ``DELTA``

``x₀`` = ``x_0``

``x₁`` = ``x_1``

``x₂`` = ``x_2`` 

``x₃`` = ``x_3`` 

``x₄`` = ``x_4`` 

``x₅`` = ``x_5`` 

``x₆`` = ``x_6`` 

``x₇`` = ``x_7`` 

``x₈`` = ``x_8``

``x₉`` = ``x_9``

``α₁`` = ``alpha_1``

``Δ`` = ``DELTA_max``
